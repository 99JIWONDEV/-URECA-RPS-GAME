# 0410 유레카 가위바위보 게임 

유레카 수업시간에 스스로 개발한 가위바위보 게임입니다. 

React의 state 개념과 props 개념을 학습했습니다.

## 프로젝트 기능 
- 컴퓨터와 가위바위보를 할 수 있습니다. 
- 한 번 가위바위보 버튼을 클릭하면 2초 동안 클릭할 수 없습니다. 
- 다시하기 버튼을 통해 초기화할 수 있습니다.
